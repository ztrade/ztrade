# AGENTS

## Documentation Language Rules

- Default language for docs is English.
- For the same document pair, keep bidirectional language switch links:
  - English doc links to its Chinese counterpart.
  - Chinese doc links to its English counterpart.
- In document indexes or navigation sections, provide both English and Chinese links so users can switch language directly.
- If code or documentation changes, always check whether related documentation must be updated.
- Always keep Chinese and English documentation content consistent when both language versions exist.
- Bilingual `SKILL` documents do not need language-switch links inside the document body.
- Exception: `doc/architecture_improvements.md` can remain single-language.
